Important things to keep in mind

...when you start working

  1. be open to unfamiliar technologies
  2. stop copy/pasting
  3. get ready for intense training
  4. work on things you like

the most frustrating part of being a junior software developer

expect this to last for the first couple of years

  1. getting a lot of negative code reviews
  2. getting questioned on code decisions
  3. not getting to code on larger features
  4. expectations are high so one week to 10 days on assignments
  5. being benched - happens when companies over recruit to show clients manpower

the most enjoyable part of being a fresher software developer

you're lucky - don't forget that

  1. learning new stuff
  2. adapting to a professional environment
  3. your growth as a human being
  4. enjoying the perks and benefits
